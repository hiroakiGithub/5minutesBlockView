# 5分ブロックビュー

時間を5分ごとのブロックに区切り、「今が何ブロック目か」を表示する時間管理アプリです。

## 使い方
`index.html` をブラウザで開くだけで動作します（インストール不要）。

- 開始時刻・終了時刻を設定すると、その範囲をブロックに分割します。
- ブロックは**毎時0分から5分刻み**のグリッドに合わせます。
- 最初の端数も**1ブロック目**として数えます。
- 設定はブラウザに自動保存されます。

## 例：9:03〜9:18 の場合
| ブロック | 範囲 |
|---|---|
| 1 | 9:03〜9:05 |
| 2 | 9:05〜9:10 |
| 3 | 9:10〜9:15 |
| 4 | 9:15〜9:18 |
